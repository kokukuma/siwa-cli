# siwa-cli

## How to use
### Apple developer center
+ Create ServiceID
+ Setting RedirectURI
  + set verification file to your domain
  + set local redirector to the redirectURI
+ Download Apple Key

### Use it
+ run redirect server 
```
go run cmd/redirector/main.go
```
### Use it
```
go run cmd/siwacli/main.go
```
